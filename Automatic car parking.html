About the Project

The Automated Car Parking System is designed to overcome parking space management issues in urban environments. Using an ESP32 microcontroller, the system intelligently monitors parking slots, guides vehicles, and improves security through automation.

The project integrates sensors, GPS modules, camera systems, and advanced user interaction methods like voice and gesture control to provide a smart parking experience.

System Architecture

The ESP32 microcontroller acts as the core processing unit. It interfaces with ultrasonic sensors, IR sensors, GPS module, camera module, and communication systems.

Sensor data is processed locally and transmitted via Wi-Fi to a monitoring dashboard, enabling real-time slot tracking and user interaction.
